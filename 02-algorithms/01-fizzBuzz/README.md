## 👽 Problem description

+ Imprimir en pantalla los números del 1 a n.
+ Si el número es divisible entre 3, imprimir Fizz.
+ Si el número es divisible entre 5, imprimir Buzz.
+ Si el número es divisible entre 3 y 5, imprimir FizzBuzz.
